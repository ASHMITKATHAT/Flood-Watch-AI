# FloodWatch Equinox Frontend

This is the frontend application for the FloodWatch Equinox system, built with React, TypeScript, and Vite.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Application pages/routes
- `src/contexts`: React Context providers (Auth, Data, Alerts)
- `src/services`: API integration services
- `src/styles`: Global and component styles
- `src/utils`: Helper functions

## Technologies
- React 18
- TypeScript
- Tailwind CSS
- Leaflet (Maps)
- Recharts (Charts)
